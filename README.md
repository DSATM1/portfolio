# DSATM1 Portfolio

A modern, responsive portfolio website showcasing projects, skills, and achievements.

## 🎨 Features

- **Responsive Design**: Mobile-first approach that works on all devices
- **Modern UI**: Clean and professional interface with smooth animations
- **Project Showcase**: Display your best projects with links to GitHub repositories
- **Skills Section**: Highlight your technical expertise
- **Smooth Scrolling**: Enhanced user experience with smooth navigation
- **Contact Links**: Easy access to social profiles and email

## 🚀 Quick Start

1. Clone or download this repository
2. Open `index.html` in your web browser
3. Customize the content with your information:
   - Update project links in the Projects section
   - Modify social links in the Contact section
   - Add your email address
   - Update descriptions and skills

## 📋 Customization Guide

### Update Personal Information
- Edit the hero section greeting in `index.html`
- Change the tagline and subtitle
- Update the About section with your bio

### Add/Modify Projects
- Find the projects grid in `index.html`
- Add new project cards with your projects
- Update GitHub repository links
- Modify tech stack badges

### Update Contact Links
- Replace LinkedIn URL in the contact section
- Update email address (change `your.email@example.com`)
- Add additional social media links

### Customize Colors
- Open `styles.css`
- Modify CSS variables in `:root`:
  ```css
  --primary: #6366f1;
  --secondary: #ec4899;
  --dark: #1f2937;
  --light: #f3f4f6;
  ```
